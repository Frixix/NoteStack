# Verificacion Manual

Checklist corta para validar la version actual de NOTESTACK.

## Cuadernos

* Crear un cuaderno nuevo
* Seleccionar otro cuaderno desde la lista
* Renombrar el cuaderno activo
* Eliminar el cuaderno activo
* Confirmar que al borrar un cuaderno tambien se borran sus notas

## Notas

* Crear una nota nueva
* Editar titulo
* Editar contenido
* Eliminar una nota
* Confirmar que al borrar una nota se selecciona otra o se limpia el editor

## Busqueda y persistencia

* Buscar por texto en titulo
* Buscar por texto en contenido
* Limpiar la busqueda y comprobar que reaparecen todas las notas del cuaderno
* Recargar la pagina y confirmar que cuadernos y notas siguen guardados

## Accesibilidad basica

* Navegar por la lista de cuadernos con `Tab`
* Activar un cuaderno con `Enter` o barra espaciadora
* Navegar por la lista de notas con `Tab`
* Activar una nota con `Enter` o barra espaciadora
